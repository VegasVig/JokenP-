# 🐉✊✋✌ JokenPô Dragon Ball Edition

Projeto web divertido desenvolvido com o tema Dragon Ball, trazendo uma versão personalizada do clássico jogo JokenPô (Pedra, Papel e Tesoura).

A aplicação combina lógica de programação, interatividade e design temático inspirado no universo Dragon Ball para proporcionar uma experiência divertida ao usuário.

## 🚀 Funcionalidades

✅ Escolha entre Pedra, Papel ou Tesoura  
✅ Jogada automática do computador  
✅ Sistema de comparação e resultado automático  
✅ Placar de vitórias, derrotas e empates  
✅ Interface personalizada com tema Dragon Ball  
✅ Design responsivo para desktop e mobile  

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto foi praticar lógica de programação, manipulação do DOM e criação de interfaces interativas de forma divertida.

Com este projeto foi possível trabalhar:

✔️ Eventos e interações do usuário  
✔️ Condições e regras lógicas  
✔️ Atualização dinâmica de elementos na tela  
✔️ Responsividade e estilização temática
